# egg-unipush

> egg plugin for unipush

## 安装

```bash
$ npm i egg-unipush --save
# or
$ yarn add egg-unipush
```

## 依赖说明

### 依赖的 egg 版本

egg 2.x | egg 1.x
--- | ---
😁 | ❌

## 开启插件

```js
// {app_root}/config/plugin.js
exports.unipush = {
  enable: true,
  package: 'egg-unipush',
};
```
